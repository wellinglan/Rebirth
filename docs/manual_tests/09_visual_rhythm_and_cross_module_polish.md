# UI-1B Visual Rhythm 与跨模块联动手动测试

1. 启动 App，检查 Today 表单、卡片和区块间距是否统一。
2. 在 Today 填写睡眠、运动和身体状态并保存。
3. 不重启 App，切换到 Health，确认今日健康数据立即更新。
4. 在 Health 修改睡眠、运动和身体状态并保存。
5. 不重启 App，切回 Today，确认健康摘要立即更新。
6. 打开 Today History，确认列表不包含今天。
7. 打开过去的 Today 记录，确认显示“已记录”而不是“草稿”。
8. 打开 Journal 最近复盘及详情，确认状态不显示“草稿”。
9. 检查 Journal 详情的问题为较粗标题，回答为常规正文，空回答显示“未填写”。
10. 查看 Health History，确认列表不包含今天，近 7 日摘要仍正常。
11. 检查 Plan 卡片、筛选条与 Dialog 仍可使用。
12. 检查 Growth、Profile 与底部导航的字体、间距和组件风格一致。
13. 执行 Today、Journal、Plan、Health 的主要保存操作，确认功能无回退。
