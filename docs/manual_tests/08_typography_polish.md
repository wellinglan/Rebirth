# Typography Polish 手动测试

1. 启动 App。
2. 查看 Today 页面标题、表单、按钮和历史入口字体是否统一。
3. 查看 Journal 页面长文本输入和历史卡片字体是否自然。
4. 查看 Plan 页面目标卡片、badge、筛选器和按钮字体是否统一。
5. 查看 NavigationBar 字体是否统一。
6. 检查中文、英文和数字混排是否自然。
7. 检查粗体是否减少，页面层级是否仍然清楚。
8. 检查表单 helper 和错误提示是否清晰可读。
9. 检查 Windows 桌面显示是否正常。
10. 确认没有字体缺失、乱码或异常 fallback。
11. 确认 Today、Journal 和 Plan 功能不受影响。
