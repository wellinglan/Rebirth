# Plan Basic UI Manual Test

1. 打开 App。
2. 进入 Plan。
3. 在空状态下点击“新建目标”。
4. 不填写 title 并尝试保存，确认显示“目标标题不能为空”。
5. 填写 title、level、status 和 targetDate，创建目标。
6. 确认目标出现在根目标列表中。
7. 点击目标进入编辑表单。
8. 修改标题和状态后保存。
9. 确认列表显示更新后的内容。
10. 通过状态菜单将目标切换为“已完成”。
11. 完全关闭 App。
12. 重新打开 App。
13. 进入 Plan 并确认目标仍然存在且状态正确。
14. 分别打开 Today 和 Journal，确认页面仍可正常加载和使用。
