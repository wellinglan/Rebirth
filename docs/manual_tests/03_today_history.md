# Today History Manual Test

1. Start the app.
2. Open Today.
3. Fill the Today form and save it.
4. Select History.
5. Confirm today's record appears in the history list.
6. Open the record and verify the read-only detail values.
7. Close the app completely.
8. Restart the app.
9. Open Today History again.
10. Confirm the saved record is still present.
11. Return to Today, change the record, and save again.
12. Open Today History and confirm there is no duplicate for today.
13. Confirm the detail shows the updated values.
14. Confirm empty fields display as not filled.
15. Confirm zero minutes displays differently from an unfilled duration.
